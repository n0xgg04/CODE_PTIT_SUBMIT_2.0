## CODE PTIT AUTO SUBMIT

@Author: [n0xgg04](https://github.com/n0xgg04)

@Version: 1.0

Chi tiết config ib hỏi LTA :)))

### Installation:

- Install package

```sh
yarn install
```

### Config

- Edit your cookie in `./common/config`
- Set up ENV in .env file

- CSRF_TOKEN, COOKIE lấy tại form submit bài trong code PTIT

### Run tool

```sh
yarn start
```
